# Data Use Note (Draft)

This repository draft includes processed, analysis-ready files and selected result tables/figures. The raw/minimal FAERS/AEMS export files used during dataset construction are not included in this draft repository.

Before public release, confirm:

1. Whether processed FAERS/AEMS-derived tables can be redistributed under the intended repository license.
2. Whether PubChem-derived identifiers and structures require source-specific attribution beyond standard citation.
3. Whether the journal requires the complete raw-processing input files, a Zenodo archive, or only processed model-ready data.
4. Whether JADER-derived data are intended to be included in this repository; no JADER model-ready files were identified in the supplied Zip.

Recommended practical approach:

- Apply an open-source license such as MIT to code.
- Apply a separate data license or data-use note to processed data.
- Cite FDA FAERS/AEMS and PubChem/NCBI in the manuscript and repository.
